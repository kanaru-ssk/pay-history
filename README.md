## Description

クレジットカードの予算管理、しっかり出来ていますか？
カード明細は反映に時間がかかってしまい、今いくら使ったのか把握するのは難しいですよね。
pay history を使ってカードの予算管理を簡単にしましょう!

## How To Use

1. 月の予算を入力する
2. カードで支払いしたら金額を入力
3. 完了!
